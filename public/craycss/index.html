Buttons

Use the class "btn" on a button tag to get a button.

Use the class "btn" on an input with type="button" to get a button.


There are different colored buttons such as "btn, btn-bad, btn-okay, btn-good, btn-dark, and btn-light"


Outline only buttons are a thing too! Use the classes inside of the buttons!

Alerts

Use the class="alert" in a div to get an alert.
class="alert">

Use these classes to change the color of the alert:

class="alert alert-default"
class="alert alert-bad"
class="alert alert-okay"
class="alert alert-good"
class="alert alert-light"


Use the avatar class to turn a regular image into an avatar:
Use the classes avatar-xl, avatar-lg, avatar-m, avatar-sm to change the sizes of the avatars respectively. Using the avatar class only will not change the size of the avatar, it will keep the image the same size! Using only an avatar size class will not turn it into an avatar, it will only change the size of the image, not round it out like an avatar.
Use the sq-avatar class instead of the avatar class to make the avatar more squared than rounded.
